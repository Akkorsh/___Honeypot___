# ___Honeypot___
## Инструкция по использованию
### Требования к системе
* Операционная система: Linux

* Язык программирования: C++ (стандарт C++11/14)

* Права доступа для создания сокетов (для портов выше 1024 достаточно прав пользователя)

### Сборка honeypot
```ruby
# Самый простой способ - компиляция напрямую
g++ -std=c++11 -o honeypot honeypot.cpp

# Запуск с выводом лога в консоль
sudo ./honeypot

# Запуск в фоновом режиме с записью лога в файл
sudo ./honeypot > honeypot.log 2>&1 &
```

### Остановка работы
Для остановки honeypot необходимо нажать сочетание клавиш _Ctrl+C_ в терминале, где запущена программа. Перед завершением будет выведена финальная статистика.
